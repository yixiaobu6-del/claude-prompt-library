# Claude Prompt 模板库

[![HTML](https://img.shields.io/badge/HTML-standalone-orange)]()
[![Claude](https://img.shields.io/badge/Claude-AI-compatible-blue)]()

Claude 提示词模板管理与构建工具。

## 功能

- **模板库**：多种场景的 prompt 模板
- **收藏管理**：localStorage 收藏常用模板
- **快速加载**：一键加载已收藏模板
- **搜索筛选**：按分类/关键词快速定位

## 使用

打开 prompt_builder.html 即可使用。
