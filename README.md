# Claude Prompt模板库

写Prompt总从头开始？
**模板库帮你快速构建高质量Prompt。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## Prompt不再"乱写"

- **模板库**：多场景Prompt模板
- **收藏管理**：localStorage收藏常用模板
- **快速加载**：一键加载已收藏模板
- **搜索筛选**：按分类/关键词定位

## 怎么用

打开 `prompt_builder.html` → 选择/搜索模板 → 复制使用
