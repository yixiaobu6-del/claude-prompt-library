# Claude Prompt 模板库

> 精选 Claude / Claude Code 提示词模板合集，覆盖多场景需求

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 写作场景模板 | 文章写作、改写、翻译、润色提示词 |
| 编程场景模板 | 代码生成、调试、重构、优化提示词 |
| 分析场景模板 | 数据分析、报告生成、决策辅助提示词 |
| 角色扮演模板 | 各领域专家模拟、对话练习提示词 |
| 结构化格式 | 每个模板含适用场景+提示词+使用技巧 |
| 可直接复制 | Markdown格式便于复制使用 |
| 持续更新 | 欢迎贡献新模板，持续扩充库内容 |

## Installation / 安装

无需安装，直接查阅模板文件。

```bash
# 克隆仓库
git clone https://github.com/yourusername/claude-prompt-template-library.git

cd claude-prompt-template-library

# 查看模板目录
ls -la
```

## Usage / 使用方法

### 基础用法

进入对应分类目录，复制 `.md` 文件中的提示词即可使用。

### 模板分类一览

| 分类 | 数量 | 说明 | 目录 |
|------|:----:|------|------|
| 写作场景 | 4 | 文案、报告、邮件模板 | `写作场景/` |
| 编程场景 | 4 | 调试、代码审查、架构模板 | `编程场景/` |
| 分析场景 | 2 | 数据分析、市场研究模板 | `分析场景/` |
| 角色扮演 | 2 | 面试官、导师、顾问模板 | `角色扮演/` |

### 模板格式示例

```markdown
## 文章润色模板

### 适用场景
当你写完文章初稿，需要优化表达和结构时使用。

### 提示词
请帮我润色以下文章，要求：
1. 保持原文核心观点不变
2. 优化语言表达，使其更流畅
3. 调整段落结构，增强逻辑性
4. 添加适当的过渡句

原文：
[粘贴你的文章]

### 使用技巧
- 可指定润色程度：轻度/中度/深度
- 可要求特定风格：学术/通俗/幽默
```

### 快速使用示例

```python
# 作为 Python 字符串使用
prompt = """
请帮我润色以下文章，要求：
1. 保持原文核心观点不变
2. 优化语言表达，使其更流畅
3. 调整段落结构，增强逻辑性

原文：
"""
prompt += your_article_text
```
