# 代码调试模板

## Bug 定位

```markdown
请帮我定位以下 Bug：
- 问题描述：{{描述 Bug 现象}}
- 期望行为：{{应该发生什么}}
- 实际行为：{{实际发生什么}}
- 相关代码：{{粘贴代码}}

分析步骤：
1. 列出可能的原因
2. 标记可疑代码行
3. 提出排查建议
```

---

## 错误日志分析

```markdown
请分析以下错误日志：
- 日志内容：{{粘贴日志}}
- 环境：{{运行环境}}
- 时间：{{发生时间}}

要求：
1. 解读错误类型
2. 定位问题代码
3. 提出修复方案
```

---

## 性能问题排查

```markdown
请分析性能问题：
- 问题现象：{{描述}}
- 运行时间：{{耗时}}
- 数据规模：{{数据量}}

要求：
1. 定位性能瓶颈
2. 分析时间复杂度
3. 提出优化方案
```

---

## 单元测试生成

```markdown
请为以下函数生成单元测试：
- 函数代码：{{粘贴代码}}
- 测试框架：{{Jest/Vitest/Mocha}}
- 测试覆盖：{{正常/异常/边界}}

要求：
1. 测试所有参数组合
2. 包含边界值测试
3. 测试异常情况
```
