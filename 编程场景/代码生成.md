# 代码生成模板

## 快速生成函数

```markdown
请生成一个函数，功能如下：
- 函数名：{{函数名}}
- 输入参数：{{参数列表}}
- 返回值类型：{{返回类型}}
- 功能描述：{{功能描述}}

要求：
1. 添加类型注释
2. 包含参数校验
3. 添加单元测试示例
```

### 示例调用

```
请生成一个函数：
- 函数名：validateEmail
- 输入参数：email: string
- 返回值类型：boolean
- 功能描述：验证邮箱格式是否合法
```

### 输出

```typescript
function validateEmail(email: string): boolean {
  if (!email || typeof email !== 'string') {
    return false;
  }
  const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return pattern.test(email);
}

// 测试示例
console.log(validateEmail('test@example.com')); // true
console.log(validateEmail('invalid')); // false
```
