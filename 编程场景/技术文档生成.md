# 技术文档生成模板

## API文档生成

```markdown
请根据以下代码生成 API 文档：
- 代码：{{粘贴代码}}
- 文档风格：{{简洁/详细/标准}}
- 目标读者：{{前端开发者/第三方开发者}}

输出格式：
1. API 概览
2. 接口列表
3. 每个接口的：
   - 请求方法 + 路径
   - 参数说明
   - 请求示例
   - 响应示例
   - 错误码说明
4. 使用注意事项
```

## SQL 优化模板

```markdown
请优化以下 SQL 查询：
- 原 SQL：{{粘贴SQL}}
- 数据规模：{{约X行}}
- 当前耗时：{{耗时}}
- 数据库类型：{{MySQL/PostgreSQL/...}}

优化方向：
1. 索引建议
2. 查询重写
3. 执行计划分析
4. 连接优化
5. 分页/分区建议

请提供优化后的 SQL 和执行效率对比。
```

## 命令行工具设计

```markdown
请帮我设计一个命令行工具：
- 工具名称：{{名称}}
- 核心功能：{{功能描述}}
- 输入参数：{{参数列表}}
- 输出格式：{{JSON/文本/表格}}
- 使用语言：{{语言}}

要求：
1. 完整的 CLI 参数解析
2. 友好的帮助信息
3. 错误处理
4. 进度显示
5. 配置文件支持（可选）

请提供完整的代码实现。
```
