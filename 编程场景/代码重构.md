# 代码重构模板

## 代码重构

```markdown
请帮我重构以下代码：
- 原代码：{{粘贴代码}}
- 重构目标：{{可读性/性能/可维护性}}
- 编程语言：{{语言}}
- 核心问题：{{问题描述}}

要求：
1. 指出原代码的问题
2. 提供重构后的代码
3. 解释重构思路和收益
4. 保持功能完全一致
```

### 重构检查清单

```
☐ 消除重复代码
☐ 简化复杂条件
☐ 提取合适的方法/函数
☐ 统一命名规范
☐ 优化数据结构
☐ 添加必要注释
```

## 代码审查

```markdown
请审查以下代码：
- 功能描述：{{功能说明}}
- 代码：{{粘贴代码}}
- 审查重点：{{安全性/性能/代码风格}}

审查维度：
1. 逻辑正确性
2. 边界情况处理
3. 性能问题
4. 安全漏洞
5. 代码可读性
6. 测试覆盖

请给每个维度打分（1-5分）并给出改进建议。
```

---

## 架构设计评审

```markdown
请评审以下系统架构设计：
- 系统目标：{{目标描述}}
- 核心模块：{{模块列表}}
- 数据流：{{数据流向}}
- 技术栈：{{技术选型}}

评审要求：
1. 评估架构的扩展性
2. 识别潜在的单点故障
3. 分析模块划分的合理性
4. 提出优化建议
```
