# 市场研究模板

## 竞品分析框架

```markdown
请帮我进行竞品分析：
- 目标领域：{{领域}}
- 主要竞品：{{竞品列表}}
- 分析维度：{{功能/定价/用户体验/市场份额}}

输出格式：
1. 竞品概览表（名称、定位、核心功能、价格）
2. 优劣势对比矩阵
3. 市场定位图
4. 差异化机会
```

---

## 行业趋势分析

```markdown
请分析以下行业趋势：
- 行业名称：{{行业}}
- 时间范围：{{过去X年}}
- 关注重点：{{技术/政策/消费/竞争}}

分析要求：
1. 识别3-5个关键趋势
2. 每个趋势的驱动力
3. 对行业格局的影响
4. 未来展望（1-3年）
```

---

## 用户调研框架

```markdown
请帮我设计用户调研方案：
- 产品/服务：{{产品描述}}
- 目标用户：{{用户画像}}
- 调研目标：{{验证假设/发现痛点/测量满意度}}

输出：
1. 调研问题清单（10-15个问题）
2. 调研方式建议（问卷/访谈/可用性测试）
3. 数据分析方法
4. 输出报告模板
```
