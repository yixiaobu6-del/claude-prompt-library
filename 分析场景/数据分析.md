# 数据分析模板

## 数据分析框架

```markdown
请帮我分析以下数据：
- 数据描述：{{数据来源和字段说明}}
- 分析目标：{{要回答的问题}}
- 分析维度：{{时间/分类/对比}}

输出格式：
1. 数据概览
2. 关键发现
3. 趋势解读
4. 建议措施
```

---

## 可视化建议

```markdown
请推荐可视化方式：
- 数据特点：{{数据类型和特征}}
- 呈现目标：{{强调趋势/对比/分布}}
- 受众：{{受众背景}}

请说明：
1. 推荐图表类型
2. 为什么适合
3. 配色建议
```

---

## 报告摘要生成

```markdown
请为以下数据生成摘要：
- 原始数据：{{粘贴数据/图表描述}}
- 重点：{{关键指标}}
- 篇幅：{{100-300字}}

要求：
1. 突出关键发现
2. 用一句话概括核心结论
3. 提供数据支撑
```

---

## 异常检测模板

```markdown
请分析以下异常数据：
- 正常范围：{{基准值}}
- 异常数据：{{异常值}}
- 时间窗口：{{观察周期}}

分析：
1. 异常程度评估
2. 可能原因
3. 是否需要告警
```
