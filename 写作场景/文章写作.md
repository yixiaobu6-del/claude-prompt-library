# 文章写作模板

## 文章框架生成

```markdown
请帮我设计文章框架：
- 主题：{{文章主题}}
- 目标读者：{{读者画像}}
- 文章类型：{{教程/观点/案例分析}}
- 预计长度：{{字数}}

要求：
1. 列出章节结构
2. 每章核心观点
3. 每章预计字数
```

---

## 引言生成

```markdown
请生成文章引言：
- 主题：{{文章主题}}
- 核心观点：{{主要论点}}
- 引入方式：{{故事/数据/提问}}
- 长度：{{200-500字}}
```

---

## 案例描写

```markdown
请扩展以下案例：
- 案例背景：{{简要描述}}
- 核心冲突：{{关键问题}}
- 解决过程：{{如何解决}}
- 最终结果：{{结局}}

要求：
1. 增加细节描写
2. 加入对话
3. 突出转折点
```

---

## 结尾升华

```markdown
请生成文章结尾：
- 核心观点：{{主要论点}}
- 结尾类型：{{总结/呼吁/反思}}
- 金句：{{是否需要金句}}

要求：
1. 回扣主题
2. 提供行动建议
3. 留有余味
```
