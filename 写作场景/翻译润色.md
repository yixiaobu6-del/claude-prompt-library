# 翻译润色模板

## 技术文档翻译

```markdown
请翻译以下技术文档：
- 原文：{{粘贴原文}}
- 目标语言：{{中文/英文}}
- 术语表：{{是否需要术语对照}}

要求：
1. 保持专业术语准确
2. 符合目标语言表达习惯
3. 保留代码块不变
```

## 文案润色

```markdown
请润色以下文案：
- 原文案：{{粘贴文案}}
- 目标风格：{{简洁/生动/专业}}

要求：
1. 提升表达流畅度
2. 增强感染力
3. 保持原意不变
```
