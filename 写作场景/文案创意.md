# 文案创意模板

## 营销文案生成

```markdown
请生成营销文案：
- 产品/服务：{{产品名称}}
- 目标用户：{{用户画像}}
- 核心卖点：{{1-3个卖点}}
- 营销渠道：{{公众号/小红书/抖音}}
- 文案风格：{{走心/硬核/幽默}}
- 行动号召：{{点击/购买/关注}}

输出要求：
1. 一个吸引眼球的标题（3个备选）
2. 开篇钩子（50字内）
3. 正文结构（300-500字）
4. 行动号召文案
5. 备选配图建议
```

---

## 社交媒体文案

```markdown
请生成社交媒体文案：
- 内容主题：{{主题}}
- 发布平台：{{小红书/抖音/微博/知乎}}
- 目标：{{涨粉/互动/转化}}
- 参考风格：{{账号名称或风格描述}}

要求：
1. 适配平台调性
2. 含话题标签建议
3. 含互动引导问题
4. 字数符合平台最佳实践
5. 提供3个不同角度版本
```

---

## 品牌故事脚本

```markdown
请帮我写一个品牌故事：
- 品牌名：{{名称}}
- 创立时间：{{时间}}
- 核心价值：{{价值观}}
- 目标用户：{{画像}}
- 关键里程碑：{{重大事件}}

故事结构：
1. 创始人为什么要做这件事
2. 遇到了什么困难
3. 如何坚持下来的
4. 现在的样子
5. 未来的愿景

请用讲故事的方式而不是罗列事实。
```
