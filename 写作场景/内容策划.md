# 内容策划模板

## 选题脑暴

```markdown
请帮我做选题脑暴：
- 领域：{{领域}}
- 目标读者：{{读者画像}}
- 内容形式：{{图文/视频/播客}}
- 本期重点：{{核心方向或关键词}}

请输出：
1. 10个选题方向（标题形式）
2. 每个选题的核心切入点
3. 预计的读者共鸣点
4. 难度评级（简单/中等/困难）
5. 推荐优先级排序
```

---

## 内容日历规划

```markdown
请帮我规划内容日历：
- 领域：{{领域}}
- 发布频率：{{每日/每周几次}}
- 平台：{{平台}}
- 下期重点：{{热点/专题/系列}}

输出格式：
| 日期 | 主题 | 形式 | 进度 | 备注 |
|------|------|------|------|------|
| 周一 | xxx | 图文 | 策划中 | 配图x3 |

要求：
1. 一周7天内容安排
2. 每天一个主要主题
3. 含1-2个热点/时效性选题
4. 含互动/粉丝参与内容
```

---

## 用户痛点调研

```markdown
请帮我分析用户痛点：
- 目标用户：{{用户画像}}
- 所属领域：{{领域}}
- 已有产品/服务：{{提供什么}}

请分析：
1. 目标用户的TOP10痛点
2. 每个痛点的紧急程度（高/中/低）
3. 每个痛点的普遍性（高/中/低）
4. 针对每个痛点的内容切入点
5. 竞品在这些痛点上的覆盖情况

分析完成后，请基于痛点优先级推荐最值得创作的3个选题。
```
