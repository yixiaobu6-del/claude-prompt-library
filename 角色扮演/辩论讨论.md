# 辩论与讨论模板

## 结构化辩论

```markdown
我们来进行一场结构化辩论：
- 辩题：{{辩题}}
- 我的立场：{{正方/反方}}
- 对方立场：{{反方/正方}}
- 节奏：你先发言，我回应，每人轮流3次

规则：
1. 每次发言不超过200字
2. 必须引用论据
3. 可以指出对方逻辑漏洞
4. 最后总结各方最佳论点
```

---

## 苏格拉底式对话

```markdown
请使用苏格拉底式提问法帮我思考：
- 主题：{{主题}}
- 我的初始观点：{{观点}}
- 我想达到的目的：{{澄清概念/检验逻辑/探索深度}}

提问方式：
1. 先问我关于这个主题的理解
2. 针对我的回答提出追问
3. 指出我思考中的盲点
4. 帮助我形成更清晰的认知
```

---

## 跨领域专家会议

```markdown
请模拟一场跨领域专家会议：
- 讨论主题：{{主题}}
- 专家阵容：{{列出3位不同领域专家}}
- 会议目标：{{目标}}

会议流程：
1. 每位专家从自己领域发表看法
2. 自由讨论环节（交叉意见）
3. 共识点总结
4. 分歧点记录
```
